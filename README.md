# Lista de Tarefas JS

Lista de tarefas criada com HTML, CSS e JavaScript para praticar manipulação do DOM, eventos, funções e armazenamento local no navegador.

O objetivo deste projeto foi construir uma aplicação simples, funcional e visualmente organizada, permitindo adicionar, concluir, remover e salvar tarefas.

## 🔗 Projeto online

Acesse o projeto publicado:

https://mthscostaz.github.io/lista-de-tarefas-js/

## 🚀 Tecnologias usadas

* HTML
* CSS

* GitHub
* GitHub Pages
* LocalStorage

## ⚙️ Funcionalidades

* Adicionar novas tarefas
* Marcar tarefas como concluídas
* Remover tarefas da lista
* Contador de tarefas totais
* Contador de tarefas concluídas
* Adicionar tarefa pressionando Enter
* Salvar tarefas no navegador com LocalStorage
* Layout responsivo para diferentes tamanhos de tela

## 📚 O que aprendi

Neste projeto, pratiquei conceitos importantes de JavaScript:

* Selecionar elementos do HTML com `document.getElementById`
* Criar eventos com `addEventListener`
* Criar funções para organizar a lógica
* Manipular listas com arrays
* Usar `map` para atualizar tarefas
* Usar `filter` para remover tarefas
* Criar elementos HTML dinamicamente com `createElement`
* Atualizar conteúdo da tela com JavaScript
* Salvar dados no navegador usando `localStorage`
* Separar estrutura, estilo e lógica em arquivos diferentes
